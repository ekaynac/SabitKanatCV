1- download the bsp according to the en715_tx2nx_bsp manual
2- unzip the file into anywhere and get the root user access : ~$ sudo -i
3- move the unzpped file to the root : ~$ mv "file name" /
4- untar the .tar.gz.001 and .tar.gz.002 : ~$ sudo cat *.tar.gz* | tar -zxvf - -i
5- go the the install folder and set up the camera and install the bsp-jetpack
6- DO NOT INSTALL ANYTHING IN A CONVENTIONAL WAY, SEARCH FOR THE BOARD SUPPORT/TX2NX SUPPORT !!
